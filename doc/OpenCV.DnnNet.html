    Sets the new input value for the network

    • blob.
    • name.
    • scalefactor.
    • mean. @see connect(String, String) to know format of the descriptor. If scale or mean values are specified, a final input blob is computed as: \f[input(n,c,h,w) = scalefactor \times (blob(n,c,h,w) - mean_c)\f]

    • scalefactor : double
    • mean : Scalar
