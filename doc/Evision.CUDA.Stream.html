    Python prototype (for reference):

    null()

    Adds a callback to be called on the host after all currently enqueued items in the stream have completed. Note: Callbacks must not make any CUDA API calls. Callbacks must not perform any synchronization that may depend on outstanding device work or other callbacks that are not mandated to run earlier. Callbacks without a mandated order (in independent streams) execute in undefined order and may be serialized.

    queryIfComplete(self)

    Returns true if the current stream queue is finished. Otherwise, it returns false.

    waitEvent(self, event)

    Makes a compute stream wait on an event.

    waitForCompletion(self)

    Blocks the current CPU thread until all operations in the stream are complete.
