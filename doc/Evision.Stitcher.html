    composePanorama(self, images)

    Positional Arguments

    • images: vector_UMat.

    Keyword Arguments

    • pano: UMat.

    These functions try to compose the given images (or images stored internally from the other function calls) into the final pano under the assumption that the image transformations were estimated before. Note: Use the functions only if you're aware of the stitching pipeline, otherwise use Stitcher::stitch. @return Status code.
