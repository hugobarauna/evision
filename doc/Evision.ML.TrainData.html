    setTrainTestSplitRatio(self, ratio)

    Positional Arguments

    • ratio: double

    Keyword Arguments

    • shuffle: bool.

    Splits the training data into the training and test parts The function selects a subset of specified relative size and then returns it as the training set. If the function is not called, all the data is used for training. Please, note that for each of TrainData::getTrain* there is corresponding TrainData::getTest*, so that the test subset can be retrieved and processed as well. @sa TrainData::setTrainTestSplit
